## GRPC Test between Java to Node.js or Electron

The Java server uses GRPC to communicate with Node.js or Electron clients.

### `./engine` Java Server

Run `RpcServer.java`

### `./node-ts` Node.js Client

Run `npm i && npm start`

### `.electron` Electron Client

Run `npm i && npm start`
